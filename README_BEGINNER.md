# 株式取引アシスタント PWA - 初心者にもわかりやすい版

楽天証券での短期株式取引をサポートする、**初心者にもやさしい**AI搭載PWAアプリです。

## 🎯 初心者向けの特徴

### 👀 一目でわかる判断表示
- **大きく「今は買い時！」「様子を見ましょう」などと表示**
- 信号機のような色分け（緑＝買い、黄＝様子見、赤＝売り）
- 絵文字で直感的に理解できる
- スコア表示で判断の強さがわかる

### 📚 専門用語を使わない説明
- 「RSI」→「買われすぎ・売られすぎ」
- 「ゴールデンクロス」→「上昇の兆し」
- すべての指標をわかりやすい日本語で説明

### 🎓 充実のヘルプ機能
- **初回起動時のチュートリアル**（スキップ可能）
- 詳しいヘルプ・用語集ページ
- 「なぜそう判断？」ボタンで理由を表示
- よくある質問への回答

### 🔄 2つの表示モード
- **かんたん表示**（デフォルト）：必要な情報だけシンプルに
- **くわしく表示**：すべての指標を見たい人向け

## 🚀 主要機能

### 1. 総合判断（0-100点）
複数の分析を総合して、わかりやすくスコア化
- 70点以上 → 買い時の可能性
- 30-70点 → 様子見
- 30点以下 → 売り時の可能性

### 2. AI価格予測
今後5日間の価格を予測し、「上がりそう」「下がりそう」を大きく表示

### 3. ニュース分析
最近のニュースが良いか悪いかを分析
- 良いニュース多い → 😊 ポジティブ
- 悪いニュース多い → 😟 ネガティブ

### 4. わかりやすいチャート
- 線が上がっていれば価格上昇
- 線が下がっていれば価格下落
- シンプルな説明付き

## 🆘 困ったときは

### よくある質問

**Q: スコアが高ければ絶対に儲かる？**
A: いいえ。スコアは参考情報です。未来は誰にもわかりません。

**Q: 銘柄コードがわからない**
A: 
- 日本株：会社名 + .T（例：7203.T = トヨタ）
- 米国株：ティッカーシンボル（例：AAPL = Apple）

**Q: どのくらいの金額から始めればいい？**
A: 数万円から始めるのがおすすめです。慣れてきたら徐々に増やしましょう。

**Q: 毎日チェックしないとダメ？**
A: いいえ。アラート機能で目標価格を設定すれば、通知が届きます。

## 📖 用語集（よく使う言葉）

| 用語 | 意味 |
|------|------|
| 株価 | 株の値段 |
| 銘柄 | 会社の株のこと |
| 買い時 | 今買うと良さそうな状態 |
| 売り時 | 今売ると良さそうな状態 |
| 様子見 | 今は動かず、待った方が良い状態 |
| 上昇トレンド | 価格が上がっている流れ |
| 下降トレンド | 価格が下がっている流れ |
| 出来高 | 取引された株の数（人気の目安） |
